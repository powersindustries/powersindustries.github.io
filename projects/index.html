TECHNOLOGIES AND TOOLS

MarkTwo - A 2D Game Engine

The MarkTwo engine is the second iteration of an ongoing 2D game engine project. Just like Tony Stark making his Iron Man suit in a cave with a “Box of scraps,” the MarkTwo engine project uses all open source libraries and dependencies to create 2D games. The engine was made for educational purposes under the MIT license.

MonoUI - UI Library for MonoGame

MonoUI is a basic UI library for the Monogame framework. Monogame is an open source, modern implementation of Microsoft’s XNA framework. Since Monogame is just a framework, it does not come with any UI types or libraries. MonoUI aims to deliver a base UI foundation for Monogame.

Fighting Games Special Moves Practicing Tool

The Fighting Game Special Moves Practicing Tool is a project aimed at helping fighting game players practice inputting special moves. For many new players, special move inputs are difficult to complete. This project is aimed at helping players get better at performing special moves.


Games

The Florida Project

The Florida Project is a 2D pixel game made using the MarkTwo engine. The game is meant to be a showcase of MarkTwo’s features rather than be a commercially viable product.

Here is an article where I talk in depth about the project.

Wizard TicTacToe

Wizard TicTacToe is traditional TicTacToe with a twist. This game plays entirely in the Windows Command Prompt giving the game a Pip Boy feeling.

Download and play the game yourself using the link here.

Set Sail

Set Sail! is a 3D endless runner game made in Unity. The player dodges incoming pieces of debris by moving right and left. Enjoy the relaxing atmosphere and survive as long as you can.
